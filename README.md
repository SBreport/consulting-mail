# Smart Branding 컨설팅 메일 생성기

구글폼 응답을 기반으로, 수강생 한 명 한 명에게 맞춤 컨설팅 메일을 자동으로 만들어 주는 웹앱입니다.
설문 응답 내용(현재 상황·수익화 방식·고민·목표 시점 등)에 따라 메일 문구가 자동으로 달라집니다.

## 🔗 접속 주소

```
https://sbreport.github.io/consulting-mail/
```

---

## ✨ 주요 기능

- **응답 맞춤 메일 자동 생성** — 설문 응답에 따라 방향 제안·시작점·솔루션·실행 플랜·후속 지원이 자동 조합됩니다.
- **1차 / 2차 전환** — 화면 상단 `[1차 발송] [2차 발송]` 버튼으로 차수를 전환합니다. 차수별로 연동 시트와 메일 문구가 다릅니다.
- **2차 대상 자동 발췌** — 2차 탭에서는 1차에 이미 컨설팅 메일을 받은 사람을 자동으로 제외하고 보여줍니다.
- **서식 포함 복사** — "✨ 서식 포함 복사" 버튼으로 복사하면, Gmail에 붙여넣을 때 제목·구분선·링크 서식이 그대로 살아납니다. (텍스트만 복사도 가능)
- **발송완료 관리** — 스프레드시트의 "발송완료" 열에 값을 입력하면 앱에서 발송완료로 분류됩니다.

---

## 🖥️ 사용 방법

1. 사이트 접속 → 상단에서 **1차 / 2차** 선택
2. 왼쪽 목록에서 **응답자 선택** → 오른쪽에 맞춤 메일이 자동 생성
3. **"✨ 서식 포함 복사"** 클릭 → Gmail 작성창에 붙여넣기(`Ctrl+V`) → 발송
4. 발송 후, 구글 스프레드시트의 **"발송완료" 열에 `1` 입력**
5. 앱에서 **"🔄 데이터 새로고침"** → 해당 응답자가 "발송완료"로 표시됨

## 🔧 자주 하는 수정

### 1) 메일 문구 수정 — `js/mail.js`

차수별로 문구가 나뉘어 있습니다.

- **`MAIL_TEMPLATES_V1`** : 1차(예전에 발송한) 문구 — 기록 보존용
- **`MAIL_TEMPLATES_V2`** : 2차(현재 발송하는) 문구 — **보통 여기를 수정**

수정할 섹션(`direction`/`startPoint`/`solution`/`actionPlan`/`closing`)에서 해당 텍스트를 고치면 됩니다.

### 2) 발신자 정보 수정 — `js/config.js`

```javascript
sender: {
    name: '최형기',
    company: '스마트브랜딩',
    phone: '010-2824-1794',
    title: '대표'
}
```

### 3) 연동 스프레드시트 변경 — `js/config.js`

`rounds` 안에서 차수별로 관리됩니다.

```javascript
rounds: {
    1: { sheetUrl: '...1차(2월) 시트 CSV...', ... },
    2: { sheetUrl: '...2차(6월) 시트 CSV...', ... }
}
```

### 4) 디자인 수정 — `css/style.css`

- 메인 색상: `#667eea` 검색해서 변경
- 글자 크기/여백: `font-size`, `padding`, `margin` 검색

---

## 📊 스프레드시트 연동 안내

- 각 시트는 **"웹에 게시 → CSV"** 로 공개된 링크를 사용합니다.
- **"발송완료" 열이 반드시 있어야** 발송 여부가 표시됩니다. (없으면 전원 "미발송"으로 보임)
  - 값: `1`(이메일), `2`(커피챗), `3`(둘 다)
- 2차 대상 발췌는 **이메일 주소**를 기준으로 1차 발송자를 제외합니다.

- Windows: `Ctrl + F5`
- Mac: `Cmd + Shift + R`
